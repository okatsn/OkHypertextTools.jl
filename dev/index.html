OkHypertextTools.render_table — Method

render_table(df; caption=" ") render df::DataFrame to a HypertextLiteral.Result.

Example

df = DataFrame( :a => 1:5, :b => 6:10)
htxt = render_table(df)
msg0 = @htl("""
    <html>

        <head>
            <style>
            h1 {
                font-size: 24px;
                font-weight: bold;
            }

            h2 {
                font-size: 18px;
                font-weight: bold;
            }

            table {
                border-collapse: collapse;
                width: 100%;
            }

            table, th, td {
                border: 1px solid black;
            }
            </style>
        </head>

        <body>
            <p>
                <p><h1> My Table: </h1></p>
                <p>$htxt</p>
            </p>
        </body>
    </html>
""")
